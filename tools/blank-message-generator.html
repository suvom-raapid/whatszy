WhatsApp Blank Message Generator

Ever wanted to send a completely empty message on WhatsApp? Normally, WhatsApp won't let you send a blank message — the send button stays grayed out. But there's a clever workaround using invisible Unicode characters. This tool generates special characters (U+200E Left-to-Right Mark) that WhatsApp treats as valid text, but display as completely invisible. The result? An empty chat bubble that surprises everyone. You can send single blank messages for pranks, or generate multiple invisible lines to create visual spacing in group chats. The blank message works on Android, iOS, WhatsApp Web, and the desktop app. Just click Generate, copy the invisible text, and paste it into any WhatsApp conversation.

How to Use This Tool

  1. Set the number of blank lines you want (1 to 100)
  2. Click the "Generate" button to create invisible characters
  3. Click "Copy" to copy the invisible text to your clipboard
  4. Open WhatsApp and paste into any chat
  5. Hit send — your contact will see an empty message bubble
