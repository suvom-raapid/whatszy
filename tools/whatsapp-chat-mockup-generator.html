How to Use This Tool

  1. Enter a contact name and online status for the chat header
  2. Select whether the message is from you (sent) or the contact (received)
  3. Type a message, optionally set the time, and click "Add Message"
  4. Repeat to build out the full conversation with multiple messages
  5. Choose light or dark WhatsApp theme, then click "Download as PNG" to save your mockup
